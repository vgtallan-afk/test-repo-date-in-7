# HTML Editor

Final carousel manager version:
- Click carousel on desktop/mobile opens Carousel Manager automatically.
- Sidebar shows all slides as thumbnails.
- Each slide has View, Replace, Up, Down, Delete.
- + Add images supports bulk upload. Each selected image becomes one slide.
- + Add empty slide.
- Remove all slides with confirmation.
- Drag images directly onto carousel to add slides.
- No visible Image 1 / Image 2 labels on the carousel itself.
- No tap-to-enlarge/lightbox.
- Mobile: one screenshot at a time, swipe left/right.
- Desktop: multiple screenshots side by side, drag/arrows/dots.
- Carousel remains resizable: width, desktop slide width, gap.
- Keeps autosave/backups and device-specific hide/delete behavior.
